## ğŸ“– How It Works

### AI Chat
The chat tutor uses `qwen/qwen3-next-80b-a3b-instruct` via the NVIDIA NIM endpoint. Every reply includes a hidden fluency score (0â€“100) + extracted vocabulary that gets saved to your local knowledge base. Future conversations use this as RAG (Retrieval-Augmented Generation) context so the AI remembers what you know.

### Quiz Generation
When you start a quiz, the AI reads your saved vocabulary and generates contextual MCQ (listening) and speaking questions tailored to your level. After you answer, the AI evaluates each response and assigns a score.

### Data Storage
All user data is stored as **plain JSON files** inside the `/data` directory. No database, no signup required. Data persists across sessions automatically.
